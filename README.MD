# Sudoku

一个用MATLAB写的数独小游戏

## 简介

数独盘面是个九宫，每一宫又分为九个小格。在这八十一格中给出一定的已知数字和解题条件，利用逻辑和推理，在其他的空格上填入1-9的数字。使1-9每个数字在每一行、每一列和每一宫中都只出现一次，所以又称“九宫格”。

## 使用方法

### GUI

```matlab
>> sudoku
```
